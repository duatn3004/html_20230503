정규식(Regular Expression) : 문자열에서 특정 내용을 찾거나 대체, 발췌하는데 사용

  • regex 단어로 많이 표현됨.
  • /regexr/i : (//) : 시작, 종료기호, (regexr) : pattern, (i) : flag
  • 정규표현식 플래그
  • i : 대/소문자 구문X
  • g : 전체 검색
  • m : 행바꿈도 포함
  • a-zA-Z : 영어 알파벳 / ㄱ-ㅎ가-힣 : 한글 / 0-9 : 숫자
  • . : 모든문자열(숫자,한글,영어,특수기호,공백 모두)줄바꿈x
  • \d : 숫자 \D : 숫자 아닌것 \s : 공백(space) \S : 공백 아닌것 \w : 영숫자 _ \w : \w 아닌것
  • 검색 패턴 : | OR a|b
  • [] : 괄호안의 문자들 중 하나 or 처리의 묶음으로 보면 됨 /abc/ => "abc"를 포함하는 /[abc]/ => "a" 또는 "b" 또는 "c"를 포함하는 [다-바] => 다,라,마,바
  • [^문자] : 괄호안의 문제를 제외한 것 [^abcd] : "a","b","c","d" 4개 문자를 제외
  • ^문자열$ : ^(시작)$(끝)
